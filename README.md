# FlowForge

A Multi-Tenant Workflow Automation Platform that enables organizations to design, execute, and monitor DAG-based workflows.

## Core Features

- Multi-Tenant SaaS Architecture
- DAG Workflow Engine
- Workflow Versioning
- Human and Automated Tasks
- Workflow Execution Monitoring
- Usage Metering
- Subscription Billing
- Event-Driven Architecture
- Microservices Architecture

## Technology Stack

- Java 21
- Spring Boot
- PostgreSQL
- Redis
- Kafka
- Docker
- Kubernetes
