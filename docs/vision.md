# Vision

## Problem Statement

Organizations frequently execute repetitive business processes such as:

- Employee Onboarding
- Leave Approval
- Purchase Requests
- Vendor Onboarding
- Incident Escalation

Many organizations lack a flexible workflow orchestration platform that allows them to define and execute custom business processes.

## Proposed Solution

FlowForge provides a multi-tenant workflow automation platform where organizations can:

- Define workflows as DAGs
- Execute workflows
- Track workflow progress
- Manage approvals
- Integrate with external systems
- Monitor workflow usage
- View subscription-based billing metrics

## Goals

- Multi-tenancy
- Workflow orchestration
- Event-driven execution
- Horizontal scalability
- Usage metering
- Subscription billing

## Non Goals

- Payment gateway processing
- BPMN compatibility
- Visual workflow designer (Phase 1)
- Low-code platform
