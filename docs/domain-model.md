# Domain Model

## Tenant

Represents an organization.

## User

Belongs to a tenant.

## Workflow Definition

Template describing a workflow.

## Workflow Version

Versioned representation of a workflow definition.

## Task Definition

Node inside a workflow.

## Workflow Instance

Runtime execution of a workflow.

## Task Instance

Runtime execution of a task.

## Usage Record

Tracks workflow and task consumption.

## Subscription Plan

Defines usage limits.

## Invoice

Generated usage summary.
