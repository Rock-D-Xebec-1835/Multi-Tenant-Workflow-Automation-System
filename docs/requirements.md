# Functional Requirements

## Tenant Management

- Create tenant
- Manage tenant users
- Assign tenant roles

## Workflow Management

- Create workflow definition
- Update workflow definition
- Version workflows
- Delete workflows

## Workflow Execution

- Start workflow
- Cancel workflow
- Retry failed tasks
- View execution status

## Metering

- Count workflow executions
- Count task executions
- Aggregate usage per tenant

## Billing

- Subscription plans
- Usage reports
- Invoice generation

# Non Functional Requirements

- Scalable
- Fault tolerant
- Multi-tenant isolation
- Event driven
- Observable
